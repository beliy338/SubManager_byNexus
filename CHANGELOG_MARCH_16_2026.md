# 📋 Журнал изменений - 16 марта 2026

## 🔧 Исправления и улучшения

### 1. 🖼️ Система логотипов сервисов
**Проблема:** Логотипы, загруженные владельцами в базу данных, не отображались. Вместо них показывались градиентные буквы.

**Решение:**
- Обновлён компонент `ServiceLogo` с автоматической обработкой ошибок загрузки
- Добавлен обработчик `onError` для изображений
- Реализован умный fallback: фото → градиентная буква при ошибке
- Добавлен фоновый цвет `bg-muted` для прозрачных изображений

**Затронутые файлы:**
- `/src/app/components/ServiceLogo.tsx` - основной компонент
- `/src/app/components/AlternativesModal.tsx` - заменён inline код на компонент
- `/src/app/components/SubscriptionInfoDialog.tsx` - заменён inline код на компонент

**Использование в приложении:**
- ✅ Dashboard (подписки пользователя)
- ✅ Analytics (графики и статистика)
- ✅ Subscriptions (управление сервисами владельцами)
- ✅ AlternativesModal (альтернативные сервисы)
- ✅ SubscriptionInfoDialog (детальная информация)
- ✅ SelectServiceModal (выбор сервиса при добавлении)

### 2. 📜 Скроллбар в чате поддержки
**Проблема:** В модальном окне поддержки отсутствовал видимый скроллбар. Пользователи не понимали, что можно прокручивать историю сообщений.

**Решение:**
- Обновлён компонент `ScrollArea` с увеличенной шириной скроллбара
- Добавлены кастомные CSS стили для видимого скроллбара
- Реализована поддержка светлой и тёмной темы
- Добавлены hover эффекты для улучшения UX
- Прокрутка колёсиком мыши работает автоматически

**Затронутые файлы:**
- `/src/app/components/ui/scroll-area.tsx` - обновлён компонент
- `/src/styles/theme.css` - добавлены кастомные стили

**Технические детали:**
```css
/* Скроллбар всегда видимый */
[data-slot="scroll-area-scrollbar"] {
  opacity: 1 !important;
  width: 12px; /* увеличена с 10px */
}

/* Адаптация под тему */
.dark [data-slot="scroll-area-thumb"] {
  background-color: rgba(139, 92, 246, 0.5); /* фиолетовый */
}

[data-slot="scroll-area-thumb"] {
  background-color: rgba(147, 51, 234, 0.4); /* оранжевый */
}
```

## 🎯 Области улучшения

### ServiceLogo компонент
**До:**
```tsx
if (logo) {
  return <img src={logo} alt={name} />
}
return <div>{name.charAt(0)}</div>
```

**После:**
```tsx
const [imageError, setImageError] = useState(false);

if (logo && !imageError) {
  return <img 
    src={logo} 
    onError={() => setImageError(true)} 
    alt={name} 
  />
}
return <div>{name.charAt(0)}</div>
```

### ScrollArea компонент
**До:**
- Скроллбар почти невидимый (opacity низкая)
- Ширина 10px
- Нет hover эффектов
- Нет адаптации под тему

**После:**
- Скроллбар всегда видимый (opacity: 1)
- Ширина 12px
- Hover эффекты с transition
- Цвета адаптируются под тему
