БЫСТРАЯ УСТАНОВКА ФЛАГОВ СТРАН В SUPABASE
==========================================

📋 ЧТО НУЖНО СДЕЛАТЬ:
1. Открыть Supabase Dashboard
2. Выполнить SQL скрипт
3. Проверить результат

🚀 ПОШАГОВАЯ ИНСТРУКЦИЯ:

ШАГ 1: Открыть SQL Editor
--------------------------
1. Перейдите на https://supabase.com
2. Войдите в свой проект
3. В левом меню найдите "SQL Editor"
4. Нажмите "+ New query"

ШАГ 2: Вставить и выполнить SQL
--------------------------------
1. Откройте файл: /insert-flags-quick.sql
2. Скопируйте весь код (Ctrl+A, Ctrl+C)
3. Вставьте в SQL Editor (Ctrl+V)
4. Нажмите "Run" (или Ctrl+Enter)
5. Дождитесь сообщения "Success"

ШАГ 3: Проверить результат
---------------------------
В SQL Editor выполните:

  SELECT * FROM country_flags ORDER BY language_code;

Должно появиться 6 записей:
✅ BY - Беларусь 🇧🇾
✅ CN - Китай 🇨🇳
✅ ES - Испания 🇪🇸
✅ FR - Франция 🇫🇷
✅ GB - Великобритания 🇬🇧
✅ RU - Россия 🇷🇺

📊 ПРОВЕРКА НА САЙТЕ:
---------------------
После выполнения SQL:
1. Откройте ваш сайт
2. Нажмите на переключатель языка (🌐 иконка)
3. Флаги должны загрузиться из базы данных
4. Если флаги не отображаются - проверьте консоль браузера (F12)

🔧 ОТЛАДКА (ТОЛЬКО ДЛЯ ВЛАДЕЛЬЦЕВ):
------------------------------------
Если вы владелец (max.sokolvp@gmail.com или belovodvadim@gmail.com):
1. Перейдите в Настройки (Settings)
2. Прокрутите вниз
3. Найдите панель "Отладка флагов"
4. Проверьте статус загрузки каждого флага

🌍 ЕСЛИ FLAGCDN.COM ЗАБЛОКИРОВАН:
----------------------------------
Если вы в России и flagcdn.com не работает:
1. Откройте файл: /alternative-flags-urls.sql
2. Выполните его в SQL Editor
3. Это переключит на FlagsAPI (работает в России)

📱 КОНТАКТЫ:
------------
Если ничего не работает:
- Проверьте подключение к Supabase
- Проверьте что URL прокси настроен (если в России)
- Проверьте консоль браузера на ошибки

✨ ГОТОВО!
----------
После выполнения всех шагов флаги будут загружаться
из базы данных и синхронизироваться на всех устройствах.
