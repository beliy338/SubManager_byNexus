## 🚀 Примеры

### Создание сервиса

```typescript
// Владелец создает Netflix

  name: "Netflix",
  category: "Кино и сериалы",
  description: "Стриминг фильмов и сериалов",

      name: "Базовый",
      price: 450,
      currency: "RUB",
      billingCycle: "monthly"
```

```typescript
// Выбирает сервис "Netflix"

// Указывает дату следующего платежа
// → Подписка создана с автоконвертацией валюты
```

```typescript
// Владелец удаляет "Netflix"
```

## 🐛 Частые проблемы

### Владелец не может создать сервис

**Решение**: Проверьте email и RLS политики

### Подписки не удаляются

**Решение**: Проверьте триггер `cascade_delete_service_subscriptions`

### Сервисы не видны

**Решение**: Проверьте политику "Anyone can view services"

**Подробнее**: `OWNERS_GUIDE.md` → Troubleshooting
