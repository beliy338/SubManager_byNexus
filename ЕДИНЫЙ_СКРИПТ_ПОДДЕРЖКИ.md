## 📋 Структура таблицы

```sql
support_messages
├── id (UUID)                    — ID сообщения
├── user_id (UUID)               — Кто отправил
├── user_email (TEXT)            — Email
├── user_name (TEXT)             — Имя
├── message (TEXT)               — Текст сообщения
├── is_admin_reply (BOOLEAN)     — Это ответ админа?
├── parent_message_id (UUID)     — ID сообщения, на которое отвечаем
├── is_read (BOOLEAN)            — Прочитано?
└── created_at (TIMESTAMP)       — Когда отправлено
```
