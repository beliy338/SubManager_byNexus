## 🚨 Частые визуальные проблемы

### Проблема 1: Сервисы не видны в каталоге

**Как выглядит:**
```
╔════════════════════════════════════╗
║  Выберите сервис из каталога   [X]║
╠════════════════════════════════════╣
║  🔍 [Поиск...]                     ║
║  📂 [Все категории ▼]              ║
║                                    ║
║  ┌──────────────────────────────┐ ║
║  │                              │ ║
║  │   Сервисы не найдены         │ ║
║  │                              │ ║
║  └──────────────────────────────┘ ║
╚════════════════════════════════════╝
```

**Решение:**
1. Обновите страницу (Ctrl+F5)
2. Проверьте консоль браузера (F12)
3. Убедитесь, что SQL выполнился успешно
4. Проверьте подключение к интернету

---

### Проблема 2: Категории не появились

**Как выглядит:**
```
📂 Категория: [Все категории ▼]

Выпадающий список:
┌─────────────────┐
│ Все категории   │
│ Кино и сериалы  │
│ Музыка          │
│ Здоровье        │
│ (нет новых!)    │
└─────────────────┘
```

**Решение:**
1. Категории добавляются автоматически из сервисов
2. Если сервисы есть, но категорий нет - очистите кеш браузера
3. Проверьте файл SelectServiceModal.tsx (строки 41-59)
4. Категории должны быть в массиве categories

---

### Проблема 3: Цены в долларах

**Как выглядит:**
```
┌──────────────────────────────┐
│ 🟡 Яндекс Плюс               │
│ от $5.99/мес          [→]   │  ← Доллары вместо рублей!
└──────────────────────────────┘
```

**Решение:**
1. Откройте Settings (⚙️)
2. Найдите раздел "Валюта"
3. Выберите **RUB (₽)**
4. Обновите страницу
