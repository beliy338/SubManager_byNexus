## 🌟 Особенности реализации

### 1. Задержка показа модалки

```typescript
setTimeout(() => {
  setShowModal(true);
}, 1500); // 1.5 секунды
```

**Почему?**
- Дает время загрузиться приложению
- Не раздражает пользователя сразу
- Улучшает UX

### 2. Флаг hasCheckedOnLogin

```typescript
const [hasCheckedOnLogin, setHasCheckedOnLogin] = useState(false);
```

**Почему?**
- Предотвращает повторный показ в одной сессии
- Если пользователь закрыл модалку - она не появится снова
- При следующем входе флаг сбросится

### 3. Проверка авторизации

```typescript
if (!user) {
  setHasCheckedOnLogin(false);
  setShowModal(false);
  return;
}
```

**Почему?**
- Модалка появляется только после входа
- На странице логина/регистрации не показывается
- Чистый UX без навязчивости
