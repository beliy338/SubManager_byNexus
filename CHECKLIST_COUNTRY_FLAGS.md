# ✅ Чеклист: Активация флагов стран

## 🎯 Цель
Решить проблему отображения флагов на опубликованном сайте

---

## 📋 Шаги активации (5 минут)

### ☐ Шаг 1: Откройте Supabase Dashboard
- Перейдите на https://supabase.com/dashboard
- Выберите ваш проект SubManager

### ☐ Шаг 2: Откройте SQL Editor
- В левом меню выберите **SQL Editor**
- Нажмите **New query**

### ☐ Шаг 3: Выполните SQL скрипт
1. Откройте файл `/supabase-country-flags-table.sql`
2. Скопируйте **весь** код (Ctrl+A → Ctrl+C)
3. Вставьте в SQL Editor (Ctrl+V)
4. Нажмите **Run** (или Ctrl+Enter)
5. Дождитесь сообщения **Success**

### ☐ Шаг 4: Проверьте таблицу
1. Откройте **Table Editor** в левом меню
2. Найдите таблицу **country_flags**
3. Убедитесь что есть **4 записи**:
   - 🇷🇺 RU (ru) - Россия
   - 🇬🇧 GB (en) - Великобритания
   - 🇧🇾 BY (be) - Беларусь
   - 🇨🇳 CN (zh) - Китай

### ☐ Шаг 5: Проверьте на сайте
1. От��ройте: https://body-order-32825369.figma.site
2. Найдите переключатель языков (🌐 + флаг)
3. Убедитесь что флаг текущего языка отображается
4. Откройте выпадающий список
5. Все 4 флага должны быть видны

---

## ✅ Проверка результата

### Флаги отображаются? ✅
- Переключатель языков показывает флаг текущего языка
- Выпадающий список показывает все 4 флага
- Флаги загружаются быстро (CDN)

### Флаги НЕ отображаются? ❌
Проверьте:
1. **SQL скрипт выполнен?** → Table Editor должен показывать таблицу
2. **4 записи в таблице?** → Должно быть RU, GB, BY, CN
3. **CDN доступен?** → Откройте https://flagcdn.com/w320/ru.png в браузере
4. **Консоль браузера** → F12 → Console (должно быть без ошибок)

## 📞 Проблемы?

### Ошибка в SQL Editor
→ Скопируйте **весь** файл `/supabase-country-flags-table.sql`

### Таблица не создается
→ Проверьте что вы выбрали правильный проект в Supabase

### Флаги не отображаются
→ Откройте консоль браузера (F12) и проверьте ошибки

### CDN недоступен
→ Используйте альтернативный CDN (см. документацию)
