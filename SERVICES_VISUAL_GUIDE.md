## 📱 Как добавить в SubManager

1. Откройте приложение SubManager
2. Нажмите **"+"** или **"Добавить подписку"**
3. Выберите **"Из каталога сервисов"**
4. Используйте фильтры:
   - 📂 По категории (Развлечения, Финансы, Шопинг)
   - 🔍 По названию
   - ⭐ Только популярные
5. Выберите сервис
6. Выберите тариф (месяц/год)
7. Укажите дату следующего платежа
8. Нажмите **"Добавить подписку"**
