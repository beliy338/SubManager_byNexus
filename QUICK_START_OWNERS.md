## Основные действия

### Для владельцев

#### ➕ Создать сервис
```
Вкладка "Подписки" → "Добавить сервис" → Заполнить форму → "Добавить"
```

#### ✏️ Редактировать сервис
```
Найти сервис в списке → Нажать 🖊️ (карандаш) → Изменить → "Сохранить"
```

#### 🗑️ Удалить сервис
```
Найти сервис → Нажать 🗑️ (корзина) → Подтвердить
⚠️ Это удалит все подписки пользователей на этот сервис!
```

### Для обычных пользователей

#### ➕ Добавить подписку
```
Вкладка "Подписки" → "Добавить подписку" → Выбрать сервис → 
Выбрать тариф → Указать дату → "Добавить"
```

## Частые вопросы

### Как добавить третьего владельца?

1. Откройте `/src/app/utils/roles.ts`
2. Добавьте email в массив:
```typescript
const OWNER_EMAILS = [
  'max.sokolvp@gmail.com',
  'belovodvadim@gmail.com',
  'новый@gmail.com'  // добавьте здесь
];
```
3. Обновите политики в Supabase (см. OWNERS_GUIDE.md)

### Что происходит при удалении сервиса?

1. Сервис удаляется из базы данных
2. Автоматически запускается триггер
3. Триггер удаляет все подписки пользователей с названием этого сервиса
4. Realtime обновления уведомляют всех пользователей
5. Подписки исчезают из их списков автоматически

### Могу ли я редактировать сервисы другого владельца?

**Да!** Оба владельца имеют одинаковые права:
- Видят все сервисы
- Могут редактировать любые сервисы
- Могут удалять любые сервисы

### Как пользователи добавляют подписки?

1. Пользователь нажимает "Добавить подписку"
2. Видит список всех сервисов, созданных владельцами
3. Может искать и фильтровать
4. Выбирает сервис и тариф
5. Указывает дату следующего платежа
6. Подписка добавляется в его список

### Нужно ли пользователям обновлять приложение?

**Нет!** Все изменения синхронизируются автоматически в реальном времени через Supabase Realtime.
