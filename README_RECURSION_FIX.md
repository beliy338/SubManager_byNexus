# 🔧 Исправление ошибки рекурсии в чате поддержки

## 🚨 Проблема

При открытии чата поддержки появляется ошибка:
```
Error loading messages: {
  "code": "42P17",
  "message": "infinite recursion detected in policy for relation \"support_messages\""
}
```

---

1. Перейдите на https://supabase.com

### 📍 Шаг 2: Выполните SQL скрипт

## 📝 Что исправлено

### Причина ошибки
Старая RLS политика делала подзапрос к той же таблице:
```sql
parent_message_id IN (
    SELECT id FROM support_messages WHERE user_id = auth.uid()
)
```
Это создавало бесконечную рекурсию.

### Решение
Создана специальная функция-помощник `is_user_parent_message()` с атрибутом `SECURITY DEFINER`, которая безопасно проверяет принадлежность сообщений без рекурсии.
