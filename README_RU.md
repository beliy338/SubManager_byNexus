# SubManager - Управление Подписками ☁️

## 🎉 Готово к использованию!

SubManager теперь использует **полное облачное хранилище Supabase** для безопасного хранения данных. Все подписки и настройки каждого пользователя автоматически сохраняются в облаке и синхронизируются между всеми устройствами.

## 🌍 Работа в России (обход блокировки)

### Проблема
Supabase заблокирован в России. Без VPN приложение не работает.

### ✅ Решение: Cloudflare Worker
Мы создали proxy-сервер на Cloudflare Workers, который обходит блокировку.

### Быстрая установка

#### Вариант 1: Через Dashboard (проще)
```
1. Откройте: https://dash.cloudflare.com/
2. Workers & Pages → Create Worker
3. Название: submanager-supabase-proxy
4. Deploy → Quick Edit
5. Скопируйте код из файла: CLOUDFLARE_DASHBOARD_SIMPLE.js
6. Save and Deploy
7. Скопируйте URL Worker
8. Создайте .env.production:
   VITE_SUPABASE_PROXY_URL=https://ваш-worker.workers.dev
```

#### Вариант 2: Через CLI (для опытных)
```bash
npm install -g wrangler
wrangler login
cd cloudflare-worker
wrangler deploy
```

### 📖 Подробные инструкции
- **Визуальный гайд:** Откройте `CLOUDFLARE_DASHBOARD_ГАЙД.html` в браузере
- **Текстовая инструкция:** `CLOUDFLARE_БЫСТРАЯ_ИНСТРУКЦИЯ.md`
- **Шпаргалка команд:** `CLOUDFLARE_КОМАНДЫ.md`
- **Решение проблем:** `БЫСТРОЕ_РЕШЕНИЕ_CLOUDFLARE.md`

### Что даёт Worker?
- ✅ Приложение работает без VPN
- ✅ Обход блокировки Supabase
- ✅ Бесплатно (100,000 запросов/день)
- ✅ Быстро (Cloudflare CDN по всему миру)

## ❓ Частые вопросы

### Безопасны ли мои данные?
✅ Да! Каждый пользователь имеет полностью изолированный аккаунт. Никто кроме вас не может видеть ваши данные.

### Нужно ли платить за облачное хранилище?
ℹ️ Зависит от настроек Supabase проекта. Обычно есть бесплатный тариф.

### Могу ли я использовать приложение оффлайн?
❌ Нет, требуется интернет-соединение для доступа к облачным данным.

### Как восстановить пароль?
⚠️ Функция восстановления пароля пока не реализована. Обратитесь к разработчикам.
