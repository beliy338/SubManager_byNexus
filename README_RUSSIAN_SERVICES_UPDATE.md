## ❓ FAQ

### Q: Что за ошибка 403?
**A:** Это ошибка при развертывании Edge Function. Она НЕ критична, т.к. приложение работает через Supabase SDK напрямую, без Edge Functions.

### Q: Нужно ли исправлять ошибку 403?
**A:** Нет. Можно игнорировать. Приложение полностью работоспособно.

### Q: Как добавить свои сервисы?
**A:** Войдите как владелец (max.sokolvp@gmail.com или belovodvadim@gmail.com) и используйте SQL:
```sql
INSERT INTO public.services (user_id, name, category, description, ...)
VALUES ((SELECT id FROM auth.users WHERE email = 'max.sokolvp@gmail.com'), ...);
```

### Q: Можно ли редактировать сервисы?
**A:** Да, только владельцы могут редактировать через SQL:
```sql
UPDATE public.services 
SET pricing_plans = ...
WHERE name = 'Яндекс Плюс';
```

### Q: Где хранятся цены?
**A:** Цены хранятся в RUB (базовая валюта). При отображении конвертируются в валюту пользователя.
