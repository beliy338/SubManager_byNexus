### Шаг 1: Откройте Supabase Dashboard

1. Перейдите на https://supabase.com
2. Войдите в свой проект SubManager
3. В боковом меню выберите **SQL Editor**

### Шаг 2: Выполните SQL скрипт

3. Вставьте код в редактор
4. Нажмите кнопку **"Run"** (или Ctrl/Cmd + Enter)
5. Дождитесь сообщения ✅ **Success**

1. В боковом меню выберите **Table Editor**

## 🎯 Почему это работает

### Проблема (ДО):
- Флаги использовали схему `figma:asset`
- ❌ Работает только локально в Figma Make
- ❌ НЕ работает на опубликованном сайте

### Решение (ПОСЛЕ):
- Флаги хранятся в Supabase Database
- ✅ URL указывают на публичный CDN (Flagpedia)
- ✅ Работает везде, включая опубликованный сайт
- ✅ Быстрая загрузка через CDN
- ✅ Автоматический fallback на резервные флаги

## 🗂️ Структура таблицы country_flags

```sql
CREATE TABLE country_flags (
  id UUID PRIMARY KEY,
  country_code TEXT NOT NULL UNIQUE,  -- RU, GB, BY, CN
  language_code TEXT NOT NULL,        -- ru, en, be, zh
  flag_url TEXT NOT NULL,             -- https://flagcdn.com/w320/...
  flag_emoji TEXT,                    -- 🇷🇺, 🇬🇧, 🇧🇾, 🇨🇳
  created_at TIMESTAMPTZ,
  updated_at TIMESTAMPTZ
);
```

**RLS Политики:**
- ✅ **Чтение**: Доступно всем (даже неавторизованным)
- 🔒 **Изменение**: Только владельцы (max.sokolvp@gmail.com, belovodvadim@gmail.com)

**Решение:**
1. Убедитесь что скопировали **ВЕСЬ** код из файла
2. Проверьте что в Supabase проект активен
3. Попробуйте выполнить скрипт по частям:
   - Сначала CREATE TABLE
   - Потом ALTER TABLE и политики
   - Затем INSERT данных

### Проблема: Флаги не отображаются на сайте

**Решение:**
1. Откройте консоль браузера (F12)
2. Проверьте наличие ошибок
3. Убедитесь что таблица `country_flags` создана
4. Проверьте что CDN доступен: откройте https://flagcdn.com/w320/ru.png

### Проблема: Ошибка доступа при INSERT

**Решение:**
- Убедитесь что вы вошли как владелец (max.sokolvp@gmail.com или belovodvadim@gmail.com)
- Или временно измените политику INSERT для тестирования:
```sql
DROP POLICY IF EXISTS "Owners can insert country flags" ON country_flags;
CREATE POLICY "Anyone can insert country flags"
  ON country_flags FOR INSERT
  WITH CHECK (true);
```
