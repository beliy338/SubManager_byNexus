# 🔧 Исправление ошибки бесконечной рекурсии

## ❌ Ошибка
```
Error: infinite recursion detected in policy for relation "support_messages"
```

## ✅ Быстрое решение

### 1️⃣ Откройте Supabase Dashboard
- Перейдите: https://supabase.com
- Выберите проект SubManager
- Откройте **SQL Editor** в левом меню

### 2️⃣ Выполните SQL скрипт
1. Создайте **New query**
2. Скопируйте содержимое файла **`fix-support-messages-recursion.sql`**
3. Вставьте в редактор
4. Нажмите **Run** (или Ctrl+Enter)

### 3️⃣ Готово!
Обновите страницу приложения - чат поддержки заработает.

---

## 📝 Что исправлено

**Проблема:**  
Старая RLS политика делала подзапрос к той же таблице, создавая бесконечную рекурсию.

**Решение:**  
Создана функция-помощник `is_user_parent_message()` с атрибутом `SECURITY DEFINER`, которая безопасно проверяет принадлежность сообщений без рекурсии.

**Результат:**
- ✅ Пользователи видят свои сообщения
- ✅ Пользователи видят ответы владельцев на их сообщения
- ✅ Владельцы (max.sokolvp@gmail.com, belovodvadim@gmail.com) видят все сообщения
- ✅ Безопасность данных сохранена
