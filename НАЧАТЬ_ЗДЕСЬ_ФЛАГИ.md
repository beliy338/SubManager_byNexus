## 📊 Структура таблицы

```sql
country_flags
├── id (UUID) - Первичный ключ
├── country_code (TEXT) - Код страны (RU, GB, BY, CN)
├── language_code (TEXT) - Код языка (ru, en, be, zh)
├── flag_url (TEXT) - URL флага из CDN
├── flag_emoji (TEXT) - Unicode emoji (🇷🇺, 🇬🇧, 🇧🇾, 🇨🇳)
├── created_at (TIMESTAMPTZ) - Дата создания
└── updated_at (TIMESTAMPTZ) - Дата обновления
```

**RLS Политики:**
- 👁️ Чтение: Доступно всем (включая анонимных)
- ✏️ Изменение: Только владельцы (max.sokolvp@gmail.com, belovodvadim@gmail.com)
