## ❓ Проблемы?

### Модальное окно не появляется?

**Причина:** `ai_permission` уже установлен

**Решение:**
```sql
-- В Supabase SQL Editor:
UPDATE user_settings 
SET ai_permission = NULL 
WHERE user_id = 'ваш-user-id';
```

### Кнопка "Анализировать" не работает?

**Причины:**
1. AI не активирован (нажмите "Да, активировать")
2. Нет подписок (добавьте хотя бы одну)
3. Не выполнен SQL скрипт

**Решение:**
1. Проверьте, что модальное окно появилось
2. Добавьте подписку
3. Выполните SQL скрипт из Шага 1

### Ошибка в консоли?

Откройте консоль браузера (F12) и проверьте:
- Есть ли красные ошибки?
- TensorFlow.js загружается?

**Типичные проблемы:**
- Старый браузер (обновите до последней версии)
- Блокировщик рекламы (отключите для localhost)
- Нет интернета при первой загрузке
