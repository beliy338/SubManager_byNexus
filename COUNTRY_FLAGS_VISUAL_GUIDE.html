Руководство по установке системы отображения флагов

❌ Проблема

Флаги не отображаются на опубликованном сайте!

При переходе на https://body-order-32825369.figma.site флаги стран не показываются, потому что они используют локальную схему figma:asset, ��оторая работает только в Figma Make.

✅ Решение

Сохранить флаги в Supabase Database!

Мы создадим таблицу country_flags в базе данных и будем загружать флаги с публичного CDN (Flagpedia), который работает везде.

1

Откройте Supabase Dashboard

Перейдите на сайт Supabase и откройте ваш проект SubManager:

https://supabase.com/dashboard

Выберите ваш проект из списка.

2

Откройте SQL Editor

В левом меню найдите раздел SQL Editor и нажмите на него.

Затем нажмите кнопку New query для создания нового запроса.

3

Вставьте SQL скрипт

Откройте файл в вашем проекте:

/supabase-country-flags-table.sql

Шаги:

  1. Откройте файл в редакторе
  2. Выделите весь текст (Ctrl+A)
  3. Скопируйте (Ctrl+C)
  4. Вставьте в SQL Editor в Supabase (Ctrl+V)
  5. Нажмите Run (или Ctrl+Enter)
4

Дождитесь успешного выполнения

После нажатия Run вы должны увидеть сообщение:

✅ Success

Это означает что таблица создана и данные добавлены!

5

Проверьте таблицу

Откройте Table Editor в левом меню.

Найдите таблицу country_flags в списке.

Убедитесь что в таблице есть 4 записи:

🇷🇺 RU → ru → https://flagcdn.com/w320/ru.png
🇬🇧 GB → en → https://flagcdn.com/w320/gb.png
🇧🇾 BY → be → https://flagcdn.com/w320/by.png
🇨🇳 CN → zh → https://flagcdn.com/w320/cn.png
6

Проверьте на сайте

Откройте опубликованный сайт:

https://body-order-32825369.figma.site

Что проверить:

  • Найдите переключатель языков (иконка 🌐 + флаг)
  • Убедитесь что флаг текущего языка отображается
  • Нажмите на переключатель
  • Все 4 флага должны быть видны в выпадающем списке

📋 Чеклист проверки

SQL скрипт выполнен без ошибок
Таблица country_flags создана
В таблице 4 записи (RU, GB, BY, CN)
