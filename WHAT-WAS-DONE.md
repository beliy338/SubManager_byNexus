## 1. Система умных уведомлений 📧

### Реализованные типы уведомлений:

**📅 Напоминания о списании**
- Email за 1-7 дней до оплаты (настраивается)
- Автоматическая проверка каждый день в 9:00 UTC
- Красивый HTML шаблон с информацией о подписке

**⚙️ Изменения в сервисах**
- Автоматическое уведомление при изменении цен/тарифов
- Срабатывает когда владелец редактирует сервис
- Отправляется всем пользователям с этой подпиской

**🚫 Прекращение работы сервиса**
- Уведомление при удалении сервиса
- Информация о необходимости найти альтернативу
- Подписки пользователей автоматически удаляются

### Где настроить:
- Откройте **Профиль** → секция "Уведомления на почту"
- Включите/выключите нужные типы уведомлений
- Выберите за сколько дней напоминать о списании

## Файлы для настройки сервера:

### 📋 Обязательно выполнить в Supabase:

1. **supabase-notifications-system.sql**
   - Создает таблицы для уведомлений
   - Настраивает триггеры и функции
   - Выполнить в Supabase Dashboard → SQL Editor

2. **supabase/functions/send-notifications/**
   - Edge Function для отправки email
   - Нужно развернуть через Supabase CLI
   - Требует API ключ от Resend

### 📖 Документация:

- **NOTIFICATIONS-QUICK-START.md** - Быстрая настройка за 5 минут
- **SETUP-NOTIFICATIONS.md** - Подробная инструкция
- **SMART-NOTIFICATIONS-README.md** - Полное описание системы

## Как начать использовать:

### Для пользователей (сразу работает в UI):
1. Войдите в приложение
2. Перейдите в **Профиль**
3. Настройте уведомления
4. Сохраните настройки ✅

### Для настройки сервера (требуется):

**Минимум (5 минут):**
1. Выполните `supabase-notifications-system.sql` в Supabase
2. Зарегистрируйтесь в [Resend](https://resend.com) (бесплатно 3000 писем/месяц)
3. Разверните Edge Function
4. Настройте Cron Job

Подробная инструкция в **NOTIFICATIONS-QUICK-START.md**
