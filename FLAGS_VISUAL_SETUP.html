🚩 Установка флагов стран в Supabase

Визуальное руководство по сохранению 6 флагов стран в базу данных

✅ Что будет работать после установки:
  • Флаги загружаются из базы данных Supabase
  • Автоматическая синхронизация между устройствами
  • Кеширование на 1 час для быстрой загрузки
  • Fallback на локальные флаги если база недоступна

2 Открыть Supabase Dashboard

  1. Перейдите на https://supabase.com
  2. Войдите в свой аккаунт
  3. Выберите проект SubManager
  4. В левом меню найдите "SQL Editor"
  5. Нажмите + New query

3 Выполнить SQL скрипт

Скопируйте и вставьте этот код в SQL Editor:

-- Создание таблицы и вставка 6 флагов
CREATE TABLE IF NOT EXISTS country_flags (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  country_code TEXT NOT NULL UNIQUE,
  language_code TEXT NOT NULL,
  flag_url TEXT NOT NULL,
  flag_emoji TEXT,
  created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

ALTER TABLE country_flags ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Anyone can view country flags"
  ON country_flags FOR SELECT USING (true);

INSERT INTO country_flags (country_code, language_code, flag_url, flag_emoji) VALUES
  ('RU', 'ru', 'https://flagcdn.com/w320/ru.png', '🇷🇺'),
  ('GB', 'en', 'https://flagcdn.com/w320/gb.png', '🇬🇧'),
  ('BY', 'be', 'https://flagcdn.com/w320/by.png', '🇧🇾'),
  ('CN', 'zh', 'https://flagcdn.com/w320/cn.png', '🇨🇳'),
  ('ES', 'es', 'https://flagcdn.com/w320/es.png', '🇪🇸'),
  ('FR', 'fr', 'https://flagcdn.com/w320/fr.png', '🇫🇷')
ON CONFLICT (country_code) DO UPDATE SET
  flag_url = EXCLUDED.flag_url,
  flag_emoji = EXCLUDED.flag_emoji,
  updated_at = NOW();
⚡ Важно:
  • Нажмите Run или Ctrl+Enter
  • Дождитесь сообщения "Success"
  • Если появилась ошибка - проверьте что скопировали весь код

5 Проверить на сайте

  1. Откройте ваш сайт SubManager
  2. Найдите переключатель языка (иконка 🌐)
  3. Нажмите на него
  4. Флаги должны загрузиться из базы данных
✅ Если всё работает:

Флаги отображаются рядом с названиями языков, и при наведении нет ошибок в консоли браузера (F12).

❓ Часто задаваемые вопросы

Флаги не отображаются на сайте?

  • Проверьте что SQL скрипт выполнился успешно
  • Откройте консоль браузера (F12) и проверьте ошибки
  • Очистите кеш браузера (Ctrl+Shift+Delete)
  • Проверьте подключение к Supabase

Как изменить размер флагов?

В URL можно изменить размер, например:

https://flagcdn.com/w320/ru.png → https://flagcdn.com/w160/ru.png

Доступные размеры: 16, 20, 24, 32, 40, 48, 64, 80, 96, 120, 160, 240, 320

Как добавить новый флаг?

INSERT INTO country_flags (country_code, language_code, flag_url, flag_emoji)
VALUES ('DE', 'de', 'https://flagcdn.com/w320/de.png', '🇩🇪');
