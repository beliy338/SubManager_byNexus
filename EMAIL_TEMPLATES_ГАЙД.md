# 📧 Макеты Email-рассылок SubManager

## 🎯 Быстрый доступ

**URL для просмотра:** `/email-templates`

---

## 📋 Два типа уведомлений

### 1️⃣ D-3: За 3 дня до списания

**📬 Тема:** "Напоминание: скоро списание по подписке {Название_сервиса}"

**🎨 Стиль:**
- Дружелюбный тон
- Фиолетовые акценты (#9333ea)
- Информативный дизайн

**🔘 CTA:**
- ✅ "Управлять подпиской"

**📝 Особенности:**
- Блок помощи "Не узнаёте сервис?"
- Возможность сообщить о проблеме
- Время для обдуманных действий

---

### 2️⃣ D-1: За 1 день до списания

**📬 Тема:** "Завтра спишем оплату за {Название_сервиса}"

**🎨 Стиль:**
- Срочный, но не агрессивный тон
- Оранжевые акценты (#fb923c)
- Дата в ярком блоке

**🔘 CTA:**
- ✅ "Проверить карту / Пополнить баланс" (основная)
- ⚠️ "Отменить подписку" (дополнительная)

**📝 Особенности:**
- Предупреждение о блокировке доступа
- Показ последних 4 цифр карты
- Акцент на срочности действия

## 🔗 URL структура

### D-3 URLs:
```
Управление подпиской:
→ /dashboard/subscriptions?id={Subscription_ID}

Сообщить о проблеме:
→ /support?subscription={Subscription_ID}
```

### D-1 URLs:
```
Проверить карту:
→ /dashboard/payment-methods?check=true

Отменить подписку:
→ /dashboard/subscriptions/{Subscription_ID}/cancel
```

## 📱 User Flow

```
1. 📧 Email получен (D-3 или D-1)
         ↓
2. 👆 Клик по кнопке в письме
         ↓
3. 🌐 Переход на сайт (с параметрами в URL)
         ↓
4. 🔐 Проверка авторизации (Supabase Auth)
         ↓
5. 📊 Открытие дашборда с нужной подпиской
         ↓
6. ⚙️ Выполнение действия:
    • Проверка карты
    • Управление подпиской
    • Отмена подписки
         ↓
7. ✅ Подтверждение (toast + email)
```

## 🛠 Техническая реализация

### Backend (Edge Function):

```typescript
// Отправка D-3 и D-1 уведомлений
// Запускается каждый день в 9:00 UTC

1. Проверить подписки с датой списания через 3 дня → отправить D-3
2. Проверить подписки с датой списания завтра → отправить D-1
3. Логировать все отправки в таблицу email_logs
```

### Frontend (React):

```typescript
// Обработка перехода из email
const [searchParams] = useSearchParams();
const subscriptionId = searchParams.get('id');

if (subscriptionId) {
  openSubscriptionModal(subscriptionId);
}
```
