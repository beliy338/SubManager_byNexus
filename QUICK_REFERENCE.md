# 🚀 Быстрая справка - SubManager

## 🖼️ Система логотипов

### Как использовать компонент ServiceLogo

```tsx
import { ServiceLogo } from './components/ServiceLogo';

// Базовое использование
<ServiceLogo logo={service.icon} name={service.name} />

// С размером
<ServiceLogo logo={sub.logo} name={sub.name} size="lg" />

// Все размеры: 'sm' | 'md' | 'lg' | 'xl'
```

### Поля в базе данных

| Таблица | Поле | Тип | Описание |
|---------|------|-----|----------|
| `services` | `icon` | text | URL логотипа (base64 или http) |
| `subscriptions` | `logo` | text | Копия из services.icon |

### Автоматический fallback

```
1. Есть URL логотипа? → Попытка загрузить
2. Загрузка успешна? → Показать фото ✅
3. Ошибка загрузки? → Показать первую букву на градиенте 🔤
4. Нет URL? → Сразу показать первую букву 🔤
```

## 🐛 Частые проблемы

### Логотип не отображается
1. Проверьте URL в базе данных
2. Убедитесь, что URL публичный
3. Проверьте формат файла (PNG, JPG, WEBP)
4. Проверьте размер файла (<500 КБ)

### Скроллбар не ��иден
1. Убедитесь, что используется компонент `ScrollArea`
2. Проверьте, что контент больше контейнера
3. Очистите кэш браузера

### Ошибка прав доступа
1. Проверьте, что пользователь авторизован
2. Проверьте email в списке владельцев
3. Проверьте RLS политики в Supabase
