## 🎯 Быстрые команды

### SQL проверка количества:
```sql
SELECT COUNT(*) FROM public.services;
```
Должно быть: **≥ 12**

### SQL проверка категорий:
```sql
SELECT category, COUNT(*) 
FROM public.services 
GROUP BY category;
```
Должны быть: **Финансы, Развлечения, Шопинг**

### SQL список российских сервисов:
```sql
SELECT name, category 
FROM public.services 
WHERE category IN ('Финансы', 'Развлечения', 'Шопинг')
ORDER BY category, name;
```
