## 🔧 Что нужно сделать администратору

### Вариант A: Отключить верификацию (для разработки) ⚡

1. Откройте https://app.supabase.com
2. Выберите проект **SubManager**
3. **Authentication** → **Settings**
4. Найдите **"Enable email confirmations"**
5. **ОТКЛЮЧИТЕ** эту опцию
6. Нажмите **Save**

✅ **Готово!** Теперь можно входить без подтверждения email.

---

### Вариант B: Настроить SMTP (для продакшена) 📧

1. **Project Settings** → **Auth** → **SMTP Settings**
2. Введите данные SMTP сервера (Gmail, SendGrid, Mailgun)
3. Настройте Sender Email
4. Сохраните изменения

✅ **Готово!** Пользователи будут получать письма с подтверждением.

---

### Вариант C: Подтвердить вручную 👤

1. **Authentication** → **Users**
2. Найдите пользователя
3. Поставьте галочку **Email Confirmed**
4. Сохраните

✅ **Готово!** Пользователь может войти.
