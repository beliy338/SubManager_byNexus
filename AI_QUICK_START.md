# ⚡ Быстрая активация AI - 30 секунд

## 🎯 1 шаг до запуска AI!

### ✅ Выполните SQL скрипт

1. Откройте https://supabase.com/dashboard
2. Выберите проект **SubManager**
3. Перейдите в **SQL Editor**
4. Нажмите **New query**
5. Скопируйте код из файла `/ADD_AI_PERMISSION_COLUMN.sql`:

```sql
-- Добавление колонки ai_permission в таблицу user_settings
ALTER TABLE user_settings 
ADD COLUMN IF NOT EXISTS ai_permission TEXT 
CHECK (ai_permission IN ('yes', 'no', 'never', 'later'));

-- Установка значения по умолчанию NULL для существующих записей
-- (NULL означает, что пользователь ещё не сделал выбор)
UPDATE user_settings 
SET ai_permission = NULL 
WHERE ai_permission IS NULL;

-- Комментарий для колонки
COMMENT ON COLUMN user_settings.ai_permission IS 
'User permission for AI features: yes (enabled), no (disabled), never (permanently disabled), later (ask later), NULL (not asked yet)';
```

6. Нажмите **RUN** (или Ctrl+Enter)
7. Должно появиться: **Success. No rows returned**

## ❓ Проблемы?

### Окно AI не появляется?

```sql
-- Проверьте значение:
SELECT ai_permission FROM user_settings WHERE user_id = 'ваш-user-id';

-- Если не NULL, установите NULL:
UPDATE user_settings SET ai_permission = NULL WHERE user_id = 'ваш-user-id';
```

### Кнопка "Анализировать" не работает?

1. Проверьте, что AI активирован (нажали "Да, активировать")
2. Добавьте хотя бы одну подписку
3. Обновите страницу (F5)

### Как найти свой user_id?

1. Откройте консоль браузера (F12)
2. Перейдите в Application → Local Storage
3. Найдите ключ с user data
4. Или посмотрите в Supabase → Authentication → Users
