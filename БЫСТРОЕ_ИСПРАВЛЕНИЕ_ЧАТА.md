3. ✅ **Счётчик красный** в **правом верхнем углу** кнопки
4. ✅ **Исправлено отображение** ответов поддержки для пользователей
5. ✅ **Исправлено отображение** сообщений владельцев в их чате

## 📝 ЧТО НУЖНО СДЕЛАТЬ

### ⚠️ ОБЯЗАТЕЛЬНО: Выполнить SQL скрипт

#### Вариант 1 (Рекомендуется) - Сохранить сообщения:
```
Файл: /fix-support-messages-view-policy.sql
```

1. Открыть **Supabase Dashboard**
2. Перейти в **SQL Editor**
3. Скопировать весь код из файла `/fix-support-messages-view-policy.sql`
4. Вставить в SQL Editor
5. Нажать **"Run"** ▶️

#### Вариант 2 - Полное пересоздание (УДАЛИТ ВСЕ СООБЩЕНИЯ!):
```
Файл: /recreate-support-messages-table.sql
```

1. Открыть **Supabase Dashboard**
2. Перейти в **SQL Editor**
3. Скопировать весь код из файла `/recreate-support-messages-table.sql`
4. Вставить в SQL Editor
5. Нажать **"Run"** ▶️

### ✅ После выполнения SQL:

1. **Перезагрузить** страницу (F5)
2. Проверить что кнопка поддержки **внизу справа**
3. Отправить тестовое сообщение
4. Войти под владельцем и ответить
5. Войти обратно под пользователем - должен появиться **красный счётчик**

## 🎨 Как выглядит

### Кнопка:
```
                    [🔴1]  ← Красный счётчик

```

### Расположение:
- **Позиция**: Правый нижний угол
- **Видимость**: На всех страницах
- **Цвет счётчика**: Красный (#EF4444)

## 🔍 Проверка

### Для пользователя:
1. ✅ Кнопка внизу справа
2. ✅ Можно отправить сообщение
3. ✅ Видны ответы от поддержки (слева, серые)
4. ✅ Счётчик горит красным когда есть новые ответы

### Для владельца:
1. ✅ Кнопка внизу справа
2. ✅ Список пользователей слева
3. ✅ Можно выбрать пользователя
4. ✅ Можно ответить на сообщение
5. ✅ Свои ответы видны в чате (справа, градиент)

## ❓ Проблемы?

### Пользователи не видят ответы поддержки:
→ Выполните SQL скрипт `/fix-support-messages-view-policy.sql`

### Владельцы не видят свои сообщения:
→ Уже исправлено в коде, просто перезагрузите страницу

### Счётчик не появляется:
→ Счётчик появляется только когда владелец **ответил** на сообщение пользователя

### Кнопки не видно:
→ Убедитесь что z-index = 50 в `/src/app/components/SupportButton.tsx`
