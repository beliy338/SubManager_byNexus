### 1. SQL скрипт поддержки - исправлены ошибки RAISE
**Файл**: `/UNIFIED_SUPPORT_SETUP.sql`

**Проблема**: Операторы RAISE NOTICE использовались вне блоков DO $$, что вызывало синтаксические ошибки в PostgreSQL.

**Решение**: 
- Все операторы `RAISE NOTICE` теперь обернуты в блоки `DO $$...END $$`
- Скрипт теперь выполняется без ошибок
- Исправлены все 4 проблемные секции:
  - После CREATE POLICY для SELECT
  - После CREATE POLICY для INSERT
  - После CREATE POLICY для UPDATE
  - После CREATE POLICY для DELETE

### 6. Разные цвета для светлой и тёмной темы
**Файл**: `/src/app/pages/Analytics.tsx`

**Проблема**: График динамики расходов использовал одинаковый фиолетовый цвет для обеих тем.

**Требование**:
- Светлая тема: фиолетовый акцент
- Тёмная тема: оранжевый акцент

**Решение**:
- Добавлен хук для отслеживания темы:
  ```typescript
  const [isDark, setIsDark] = React.useState(false);
  
  React.useEffect(() => {
    const checkTheme = () => {
      setIsDark(document.documentElement.classList.contains('dark'));
    };
    checkTheme();
    
    const observer = new MutationObserver(checkTheme);
    observer.observe(document.documentElement, {
      attributes: true,
      attributeFilter: ['class']
    });
    
    return () => observer.disconnect();
  }, []);
  ```
- График использует динамический цвет:
  - Светлая тема: `#8b5cf6` (фиолетовый)
  - Тёмная тема: `#fb923c` (оранжевый)
- Цвета в топе подписок также адаптируются: `text-orange-500 dark:text-orange-400`
