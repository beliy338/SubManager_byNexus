## 🧪 Тестирование

### Сценарии для проверки:

1. **Email верификация отключена:**
   - Регистрация → Автоматический вход → Dashboard

2. **Email верификация включена:**
   - Регистрация → Сообщение "Проверьте почту" → Перенаправление на Login
   - Вход без подтверждения → Ошибка "Email не подтвержден" → Кнопка "Отправить повторно"
   - Клик на кнопку → Успешное сообщение → Скрытие через 5 сек

3. **Неверные данные:**
   - Неверный пароль → Ошибка "Неверный email или пароль"
   - Пустой email при повторной отправке → Ошибка "Введите email"
