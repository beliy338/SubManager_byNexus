# Исправление ошибки "Email not confirmed"

## Проблема
При попытке входа в систему появляется ошибка: `AuthApiError: Email not confirmed`

## Причина
В настройках Supabase включена обязательная верификация email, но пользователи пытаются войти с неподтвержденными адресами.

## Решение

### Вариант 1: Отключить обязательную верификацию email (для разработки)

1. Откройте панель управления Supabase: https://app.supabase.com
2. Выберите ваш проект SubManager
3. Перейдите в **Authentication** → **Settings** (или **Authentication** → **Email Auth**)
4. Найдите опцию **"Enable email confirmations"** или **"Confirm email"**
5. **Отключите** эту опцию
6. Сохраните изменения

После этого пользователи смогут входить в систему сразу после регистрации без подтверждения email.

### Вариант 2: Настроить Email Provider (для продакшена)

Если вы хотите использовать верификацию email, нужно настроить SMTP:

1. Откройте панель управления Supabase
2. Перейдите в **Project Settings** → **Auth**
3. Прокрутите до раздела **SMTP Settings**
4. Настройте SMTP сервер (например, Gmail, SendGrid, Mailgun)
5. Введите данные:
   - SMTP Host
   - SMTP Port
   - SMTP Username
   - SMTP Password
   - Sender Email
6. Сохраните настройки

После настройки SMTP пользователи будут получать письма с подтверждением на указанный email.

### Вариант 3: Вручную подтвердить email существующих пользователей

Если у вас уже есть зарегистрированные пользователи с неподтвержденными email:

1. Откройте панель управления Supabase
2. Перейдите в **Authentication** → **Users**
3. Найдите нужного пользователя
4. Нажмите на него для редактирования
5. Найдите поле **Email Confirmed** и установите галочку
6. Сохраните изменения

## Использование

### Для пользователей с неподтвержденным email:

1. Попытайтесь войти в систему
2. Увидите сообщение: "Email не подтвержден. Проверьте вашу почту."
3. Нажмите кнопку **"Отправить письмо повторно"**
4. Проверьте вашу почту
5. Перейдите по ссылке подтверждения
6. Войдите в систему

### Для новых пользователей:

1. Зарегистрируйтесь на странице регистрации
2. Увидите сообщение: "Аккаунт успешно создан! Проверьте вашу почту для подтверждения email."
3. Проверьте вашу почту
4. Перейдите по ссылке подтверждения
5. Автоматически перенаправит на страницу входа
6. Войдите с вашими учетными данными
