# 🚀 Быстрый старт - SubManager

## Добро пожаловать в SubManager!

Теперь все ваши данные хранятся в облаке и синхронизируются между устройствами.

---

## 📱 Как начать использовать

### Вариант 1: Демо-доступ (быстрый тест)
```
1. Откройте приложение
2. Нажмите кнопку "🔑 Демо-вход"
3. Начните добавлять подписки
```
**⚠️ Внимание:** Демо-аккаунт использует локальное хранилище. Данные не синхронизируются между устройствами.

**Учётные данные для демо:**
- Email: `demo@submanager.app`
- Password: `админ0`

---

### Вариант 2: Создание реального аккаунта (рекомендуется)
```
1. Откройте приложение
2. Нажмите "Зарегистрироваться"
3. Заполните форму:
   - Имя
   - Email
   - Пароль (минимум 6 символов)
4. После регистрации вы автоматически войдёте в систему
```

**✅ Преимущества:**
- Данные хранятся в облаке
- Синхронизация между всеми устройствами
- Безопасное хранение
- Доступ с любого устройства

---

## 💡 Основные функции

### 1. Добавление подписки
```
Dashboard → Кнопка "Добавить подписку" → Заполнить форму → Сохранить
```

### 2. Просмотр всех подписок
```
Перейти в раздел "Подписки" в боковом меню
```

### 3. Аналитика
```
Перейти в раздел "Аналитика"
- Общие расходы
- Расходы по категориям
- Прогноз на год
```

### 4. Настройки
```
Перейти в раздел "Настройки"
- Сменить тему (тёмная/светлая)
- Выбрать язык (RU/EN/BY/CN)
- Выбрать валюту (RUB/USD/EUR/CNY/BYN)
- Изменить размер текста
```

## 🔐 Безопасность

### Ваши данные защищены:
- ✅ Шифрование при передаче (HTTPS)
- ✅ Изоляция данных между пользователями
- ✅ JWT токены для авторизации
- ✅ Безопасное хранение паролей (хэширование)

### Что НЕ хранится на сервере:
- ❌ Пароли в открытом виде
- ❌ Данные других пользователей (полная изоляция)

## 🆘 Часто задаваемые вопросы

### 1. Я потерял доступ к аккаунту. Что делать?
```
Сейчас восстановление пароля не реализовано.
Обратитесь в поддержку или создайте новый аккаунт.
```

### 2. Мои данные пропали после выхода!
```
Если вы использовали демо-аккаунт, данные хранятся локально.
Создайте реальный аккаунт для облачного хранения.
```

### 3. Как удалить аккаунт?
```
Сейчас эта функция не реализована.
Просто перестаньте использовать приложение.
```

### 4. Можно ли использовать на нескольких устройствах?
```
Да! Для реальных аккаунтов данные синхронизируются автоматически.
Просто войдите с теми же учётными данными.
```

### 5. Где хранятся мои данные?
```
В облачной базе данных Supabase (PostgreSQL).
Сервера находятся в защищённых дата-центрах.
```

## 🛠️ Возможные проблемы

### Проблема: "Unauthorized" при входе
**Решение:**
```
1. Проверьте правильность email и пароля
2. Убедитесь, что аккаунт был создан
3. Попробуйте очистить кэш браузера
```

### Проблема: Подписки не сохраняются
**Решение:**
```
1. Проверьте интернет-соединение
2. Обновите страницу
3. Попробуйте выйти и войти снова
```

### Проблема: Настройки не применяются
**Решение:**
```
1. Обновите страницу
2. Проверьте консоль браузера на ошибки (F12)
3. Очистите кэш
```
