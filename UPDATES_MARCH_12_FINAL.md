## Требуется ручная настройка

### ⚠️ Создание таблицы support_messages

**ВАЖНО:** Необходимо вручную создать таблицу в Supabase!

Следуйте инструкциям в файле `/SUPPORT_TABLE_INSTRUCTIONS.md`:

1. Откройте Supabase Dashboard
2. Перейдите в SQL Editor
3. Выполните SQL-скрипт из файла инструкций
4. Проверьте создание таблицы и политик RLS

**Таблица не будет работать без этого шага!**
