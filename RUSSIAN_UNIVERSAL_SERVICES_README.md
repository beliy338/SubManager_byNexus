# Добавление российских универсальных подписок

## 📋 Обзор

Этот скрипт добавляет 12 популярных российских универсальных подписок в базу данных SubManager. Включены подписки операторов связи, финтех-сервисов и маркетплейсов.

## 🚀 Инструкция по применению

### Шаг 1: Подготовка
Убедитесь, что:
- ✅ Таблица `services` уже создана (используйте `/supabase-services-table.sql`)
- ✅ У вас есть доступ к Supabase SQL Editor
- ✅ Владелец max.sokolvp@gmail.com зарегистрирован в системе

### Шаг 2: Выполнение скрипта
1. Откройте Supabase Dashboard
2. Перейдите в **SQL Editor**
3. Скопируйте содержимое файла `/supabase-insert-russian-universal-services.sql`
4. Вставьте в редактор и нажмите **Run**

### Шаг 3: Проверка
```sql
-- Проверить количество добавленных сервисов
SELECT COUNT(*) FROM public.services 
WHERE name IN (

    'Газпром Бонус', 'Халва.Десятка', 'Tinkoff Pro',

    'OZON Premium', 'Х5 Клуб Пакет'
);
-- Результат должен быть: 12

-- Посмотреть все добавленные сервисы
SELECT name, category, pricing_plans->>0->>'price' as price 
FROM public.services 
ORDER BY name;
```

## 📊 Структура данных

Каждый сервис содержит:
- **name** - Название сервиса
- **category** - Категория (Развлечения, Финансы, Шопинг)
- **description** - Описание возможностей
- **icon** - Эмодзи-иконка
- **is_popular** - Флаг популярности (все `true`)
- **pricing_plans** - JSON с тарифами:
  ```json
  {
    "name": "Базовый",
    "price": 399,
    "currency": "RUB",
    "billingCycle": "monthly",
    "features": ["Функция 1", "Функция 2"]
  }
  ```

## 🔄 Обновление данных

Если нужно обновить сервис:

```sql
UPDATE public.services 
SET 
    pricing_plans = '[{"name":"Новый тариф","price":499,"currency":"RUB","billingCycle":"monthly"}]'::jsonb,
    updated_at = NOW()
WHERE name = 'Название сервиса';
```
