### Row Level Security (RLS)

Политики доступа:
```sql
-- Просмотр: все аутентифицированные пользователи
"Anyone can view services"

-- Создание: только владельцы
"Only owners can insert services"
  WHERE email IN ('max.sokolvp@gmail.com', 'belovodvadim@gmail.com')

-- Обновление: только владельцы
"Only owners can update services"

-- Удаление: только владельцы
"Only owners can delete services"
```

### Cascade Delete

При удалении сервиса автоматически удаляются все подписки пользователей на этот сервис:
```sql
CREATE TRIGGER cascade_delete_service_subscriptions
```
