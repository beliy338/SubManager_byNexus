# Интеграция YandexGPT в SubManager

## 📊 Лимиты бесплатного тарифа YandexGPT

### Trial период (бесплатно)
- **1000 запросов в день**
- Доступ ко всем моделям
- До 8000 токенов на запрос

### После trial (Freemium)
- **Первые 1000 запросов в месяц бесплатно**
- Далее платно: ~0,3₽ за 1000 токенов
- Модели: yandexgpt-lite (быстрая) и yandexgpt (полная)

## 🚀 Как получить API ключ

### Шаг 1: Регистрация в Yandex Cloud
1. Перейдите на https://cloud.yandex.ru
2. Зарегистрируйтесь или войдите
3. Активируйте trial (60 дней бесплатно)

### Шаг 2: Создание сервисного аккаунта
1. Откройте консоль Yandex Cloud
2. Выберите ваш каталог (folder)
3. Перейдите в раздел "Сервисные аккаунты"
4. Создайте новый сервисный аккаунт
5. Выдайте ему роль `ai.languageModels.user`

### Шаг 3: Получение API ключа
1. Откройте созданный сервисный аккаунт
2. Нажмите "Создать новый ключ"
3. Выберите "API-ключ"
4. Скопируйте ключ (он показывается только один раз!)

### Шаг 4: Получение Folder ID
1. В консоли Yandex Cloud откройте ваш каталог
2. В правом верхнем углу найдите ID каталога
3. Скопируйте его (формат: b1g1234567890abcdefg)

## 🔧 Настройка в SubManager

1. Откройте **Настройки** в приложении
2. Прокрутите до раздела **API ключи YandexGPT**
3. Нажмите **"Показать ключи"**
4. Вставьте **API ключ** и **Folder ID**
5. Нажмите **"Сохранить"**

## ❓ FAQ

**Q: Нужно ли платить за YandexGPT?**  
A: Первые 1000 запросов в месяц бесплатно. Для большинства пользователей этого достаточно.

**Q: Что делать, если закончились бесплатные запросы?**  
A: Приложение автоматически переключится на локальный TensorFlow.js или можно пополнить баланс Yandex Cloud.

**Q: Безопасно ли передавать данные в YandexGPT?**  
A: Да, YandexGPT не сохраняет ваши запросы и соответствует стандартам безопасности.

**Q: Можно ли использовать без API ключа?**  
A: Да, приложение работает с локальным TensorFlow.js в режиме без ключа.
