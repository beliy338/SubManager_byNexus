# 🚀 Quick Start: Email Templates

## Что это?

Готовые макеты триггерных email-рассылок для SubManager:
- **D-3**: За 3 дня до списания (информативное напоминание)
- **D-1**: За 1 день до списания (срочное уведомление)

URL: **`/email-templates`**

## 📱 Что включено?

### 4 вкладки:

1. **D-3 Template** 
   - Превью шаблона за 3 дня
   - Тема: "Напоминание: скоро списание..."
   - Дружелюбный дизайн

2. **D-1 Template**
   - Превью шаблона за 1 день  
   - Тема: "Завтра спишем оплату..."
   - Срочный дизайн

3. **Сравнение**
   - Side-by-side сравнение обоих шаблонов
   - Временная шкала отправки
   - Ключевые различия

4. **User Flow**
   - Схема: Email → Dashboard → Action
   - Технические детали
   - Плейсхолдеры для backend

## 🔑 Ключевые плейсхолдеры

```
{Имя_пользователя}    → "Иван"
{Название_сервиса}    → "Яндекс Плюс"
{Сумма}               → 399
{Валюта}              → "₽"
{Дата_списания}       → "19 марта 2026"
{Subscription_ID}     → "abc-123..."
{Последние_4_цифры}   → "1234"
```

## 🎯 Для backend-разработчиков

### Краткая инструкция:

1. **Создать Edge Function**
   ```typescript
   // Отправка D-3 и D-1 писем
   // Запуск: каждый день в 9:00 UTC
   ```

2. **Настроить Cron Job**
   ```sql
   SELECT cron.schedule('send-emails', '0 9 * * *', ...);
   ```

3. **Логировать отправки**
   ```sql
   CREATE TABLE email_logs (...);
   ```

4. **Подключить email-сервис**
   - Рекомендуется: Resend или SendGrid
   - API key в переменные окружения
